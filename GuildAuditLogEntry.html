Represents a guild audit log entry describing a moderation action

Name Type Attributes Description
id String The ID of the entry
guild Guild The guild containing the entry
actionType Number The action type of the entry. See Constants.AuditLogActions for more details
reason String <nullable>
 The reason for the action
user User The user that performed the action
targetID String The ID of the action target
target CategoryChannel | Guild | Member | Invite | Role | Object | TextChannel | VoiceChannel <nullable>
 The object of the action target If the item is not cached, this property will be null If the action targets a guild, this could be a Guild object If the action targets a guild channel, this could be a CategoryChannel, TextChannel, or VoiceChannel object If the action targets a member, this could be a Member object If the action targets a role, this could be a Role object If the action targets an invite, this is an Invite object If the action targets a webhook, this is null If the action targets a emoji, this could be an emoji Object If the action targets a message, this is a User object
before Object <nullable>
 The properties of the targeted object before the action was taken For example, if a channel was renamed from #general to #potato, this would be `{name: "general"}``
after Object <nullable>
 The properties of the targeted object after the action was taken For example, if a channel was renamed from #general to #potato, this would be `{name: "potato"}``
count Number <nullable>
 The number of messages deleted, action type 72 (MESSAGE_DELETE) only
channel CategoryChannel | TextChannel | VoiceChannel <nullable>
 The channel containing the deleted messages, action type 72 (MESSAGE_DELETE) only
deleteMemberDays Number <nullable>
 The number of days of inactivity to prune for, action type 21 (MEMBER_PRUNE) only
membersRemoved Number <nullable>
 The number of members pruned from the server, action type 21 (MEMBER_PRUNE) only
member Member | Object <nullable>
 The member described by the permission overwrite, action types 13-15 (CHANNEL\_OVERWRITE\_CREATE/UPDATE/DELETE) only. If the member is not cached, this could be {id: String}
role Role | Object <nullable>
 The role described by the permission overwrite, action types 13-15 (CHANNEL\_OVERWRITE\_CREATE/UPDATE/DELETE) only. If the role is not cached, this could be {id: String, name: String}
